Holy Grail Layout

The Holy Grail Layout is a classic CSS problem with various solutions presented over time. If you're unfamiliar with the history of the Holy Grail layout, this A List Apart article offers a pretty good summary and links to a few of the more well-known solutions.

At its core, the Holy Grail Layout is a page with a header, footer, and three columns. The center column contains the main content, and the left and right columns contain supplamental content like ads or navigation.

Most CSS solutions to this problem aim to meet a few goals:

  • have a fluid center with fixed-width sidebars
  • the center column (main content) should appear first in the HTML source
  • all columns should be the same height, regardless of which column is actually the tallest
  • require minimal markup
  • have the footer "stick" to the bottom of the page if there's not enough content

However, because of the nature of these goals and the original limitations of CSS, none of the classic solutions to this problem were ever able to meet all of them.

With Flexbox, however, a solution that meets all of these goals is trivial.

<body class="HolyGrail">
  <header>...</header>
  <div class="HolyGrail-body">
    <main class="HolyGrail-content">...</main>
    <nav class="HolyGrail-leftSidebar">...</nav>
    <aside class="HolyGrail-rightSidebar">...</aside>
  </div>
  <footer>...</footer>
</body>

Geting the center content row to stretch and the footer to stick to the bottom is solved in the same was as in the Sticky Footer example.The only difference in the center row (.HolyGrails-body) needs to be display: flex as well to arrange its children.

.HolyGrail {
  display: flex;
  min-height: 100vh;
  flex-direction: column;
}

.HolyGrail-body {
  display: flex;
  flex: 1;
}

Three equal-height columns with a fluid center and fixed-width sidebars is just as easy:

.HolyGrail-content {
  flex: 1;
}

.HolyGrail-leftSidebar, .HolyGrail-rightSidebar {
  /* 12em is the width of the columns */
  flex: 0 0 12em;
}

.HolyGrail-leftSidebar {
  /* put the nav on the left */
  order: -1;
}
