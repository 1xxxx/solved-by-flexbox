Sticky Footer

Click the button below to hide the contents of this page. Notice how the footer sticks to the bottom of the window even when there's not enough content to fill the page.

Geting your footer to sticky to the bottom of a page with sparse content is something every Web developer has probably tried to tackle at some point in their career. And, for the most part, it's a solved problem. Yet all the existing solutions have one significant shortcoming — they don't work if the height of your footer can change.

Flexbox solves this problem and is really an idea solution for it. Flexbox is known for helping with layout in the horizontal direction, but it's just as good at helping with layout in the vertical direction. All you have to do is wrap the vertical sections in a flex container and choose which ones you want to flex. They'll automatically take up all the available space.

The HTML

<body class="Site">
  <header>...</header>
  <main class="Site-content">...</main>
  <footer>...</footer>
</body>

The CSS

.Site {
  display: flex;
  min-height: 100vh;
  flex-direction: column;
}

.Site-content {
  flex: 1;
}
