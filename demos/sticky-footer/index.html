Sticky Footer

Click the button below to hide the contents of this page. Notice how the footer sticks to the bottom of the window even when there's not enough content to fill the page.

Getting the footer to stick to the bottom of pages with sparse content is something just about every Web developer has tried to tackle at some point in their career. And, for the most part, it's a solved problem. Yet all the existing solutions have one significant shortcoming — they don't work if the height of your footer can change.

Flexbox is really a perfect fit for this type of problem. While mostly known for laying out content in the horizontal direction, Flexbox actually works just as well for vertical layout problems. All you have to do is wrap the vertical sections in a flex container and choose which ones you want to expand. They'll automatically take up all the available space.

The HTML

<body class="Site">
  <header>...</header>
  <main class="Site-content">...</main>
  <footer>...</footer>
</body>

The CSS

.Site {
  display: flex;
  min-height: 100vh;
  flex-direction: column;
}

.Site-content {
  flex: 1;
}
